# Calculator App

A simple yet visually appealing calculator app built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Additional functionalities: percentage, clear display, delete last entry.
- Responsive design with a modern look.
- Smooth transitions and interactions.

## Getting Started

These instructions will help you set up and run the calculator app on your local machine.

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Safari).

## File Structure

- `index.html`: The main HTML file containing the structure of the calculator.
- `styles.css`: The CSS file that styles the calculator, making it visually appealing.
- `script.js`: The JavaScript file that adds functionality to the calculator.

## Usage

- Click on the buttons to input numbers and perform operations.
- The display shows the current input or result.
- Use the `C` button to clear the display.

- Use the `=` button to calculate the result.
